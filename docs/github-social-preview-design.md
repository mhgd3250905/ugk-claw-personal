# GitHub Social Preview Design

## Quiet Cockpit

这张 social preview 采用“Quiet Cockpit”的视觉方向：黑色工作台作为底层空间，细网格与低亮度光带表达 agent runtime 的工程秩序，中心品牌图形提供第一眼记忆点，右侧用极少文字说明项目价值。

构图重点不是堆功能，而是把“自托管 agent cockpit”这个概念做成一张克制、可识别、技术感足够的横版封面。色彩以深空黑、冷白、石墨灰和一条低饱和青紫光带为主，避免变成廉价渐变海报。

文字只保留三层：品牌名、定位短句、能力标签。信息通过空间层级和对齐关系表达，避免把 README 第一段搬到图片里。最终用途是 GitHub repository social preview，尺寸固定为 `1280x640`。
